How to bulid a blog using GitHub Pages

Posted by JYL on December 14, 2020

Official Guide

Go Go Go

Steps

一：进入GitHub,新建一个名字为“username.github.io”的repository,其中，username是你的GitHub用户名。 如下图所示。（由于我已经创建过同名repository，所以图中报错。）

二：打开终端，输入命令，记得将两处username替换为你的GitHub用户名
git clone https://github.com/username/username.github.io

三：在终端进入到项目文件夹中，替换username为你的GitHub用户名
cd username.github.io

四：在终端输入以下代码，创建测试文件
echo "Hello World" > index.html

五：将文件夹中到内容add,commit并push到GitHub
git add --all
git commit -m "Initial commit"
git push -u origin master

六：打开浏览器输入如下网址（记得替换username）,个人博客就大功告成了！
https://username.github.io.
